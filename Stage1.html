Projet de Stage 1

Création d'une application avec le framework de Java : Spring, permettant de récupérer certaines parties d'un fichier présent sur un serveur distant et de les
ranger dans une base de données Oracle prévu à cette effet. Les données prélevés vont permettre d'identifier les différentes erreurs qui peuvent
exister mais qui ne sont renvoyé qu'au serveur distant. On pourra également
connaître l'heure à laquelle chacune est arrivée et donc par conséquent leurs fréquence. (utilisation de java et de son framework Spring, Oracle et regex)
